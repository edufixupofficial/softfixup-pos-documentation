Core Features

This is a complete POS template built with React and Bootstrap 5 with Rest API integration. This template is backend ready and it's built with node, express, and uses PostgreSQl for the database. We did all schema validation for saving data.
Here is some core feature of this project
  • Complete Full-Stack Project with backend integration.
  • Dynamic translation on all dynamic data.
  • Dynamic dashboard with (Overview of day/month/all sold, purchases, expense, profit, total receivable & Payable)
  • Multi Store Management.
  • Send Promotional Message to customer
  • Easy POS
  • Auto Barcode Genaretor
  • Custom Unit Add
  • Brand & Category wise Product Add
  • Multi payment Options
  • Customers Management and Suppliers Management
  • Sales and Returns.
  • Purchases & Return.
  • SR Management
  • Reports
  • Expense Management
  • Data Backup
  • Backend use TypeScript

Installation

To run react project you will need couple thing to setup first, Install Node. if you already not installed.

NPM Packages:
  • After downloading the template, unzip it
  • Open the VS Code Terminal/System Terminal
  • Go to the folder (backend): (cd path/to/template)
  • Run: npm install, it will install all used packages for this backend.
  • Make sure that there is no error.

After the npm packages are installed, make sure that you have a package.json file and check that you have the below lines under the scripts. 

        
          "scripts": {
            "dev": "ts-node-dev --respawn --transpile-only src/server.ts",
            "lint:check": "eslint --ignore-path .eslintignore --ext .js,.ts .",
            "lint:fix": "eslint . --fix",
            "build": "tsc",
            "prettier:check": "prettier --ignore-path .gitignore --write \"/*.+(js|ts|json)\"",
            "prettier:fix": "prettier --write .",
            "lint-prettier": "yarn lint:check && yarn prettier:check",
            "postinstall": "prisma generate",
            "type:check": "tsc --noEmit",
            "migrate": "npx prisma migrate dev",
            "generate": "npx prisma generate",
            "studio": "npx prisma studio"
          }
  1. Rename .env.example to .env
  2. Configure your PostgreSQl database. Watch this video on PostgreSQl . After configuring, you will find a PostgreSQl URI. Put that URI in the DATABASE_URL variable in your .env file.
  3. The JWT_SECRET is a random value used for creating a secret token. You can use any value, but make sure it remains secret.
  4. For sending Promotional SMS to Customers Yo need to add SMS API
After configuration you .env file will look like this 
                
                  NODE_ENV=development
                  PORT=5001
                  DATABASE_URL='postgres://postgres.password:databaseName@aws-0-ap-southeast-1.pooler.supabase.com:5432/postgres'
                  SMS_BASE_URL="your SMS_BASE_URL"
                  JWT_SECRET= '###AAAaaa'
                  JWT_EXPIRES_IN="90d"
                  SR_DEFAULT_PASSWORD="123456"
Running the Backend

Now run npm run dev to start your backend on the local server with PORT 5001 or the PORT you have specified.

Run On Local server

To run on local server, first you will need to configure backend

After properly configuration open the root folder on VS code or any other code editor. split the Terminal by clicking on [|] button and make into three, then go to backend by cd folder_name.
And finally run npm run dev for backend and frontend
Then frontend will run on http://localhost:3000/
and backend will http://localhost:5001 and your specified PORT.
